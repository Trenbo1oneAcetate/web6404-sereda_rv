# Лабораторная работа №3

## Дисциплина
Web-программирование

## Тема
Изучение основ JavaScript.

## Цель работы
Изучить основы языка JavaScript, принципы работы с функциями,
циклами, рекурсией, замыканиями, массивами и объектами.

## Задание
Реализовать набор функций на языке JavaScript в соответствии с
выданным каркасом проекта. Корректность реализации проверяется
с помощью автоматических тестов.

## Описание решения
В ходе выполнения лабораторной работы были реализованы функции,
выполняющие различные вычисления и проверки:
- проверка целых чисел;
- работа с массивами;
- вычисление суммы, факториала и чисел Фибоначчи;
- проверка степени двойки;
- использование замыканий;
- генерация арифметической последовательности;
- рекурсивное сравнение объектов.

Все функции реализованы в файле `src/core.js`.
Корректность работы подтверждена прохождением всех тестов.

## Используемые технологии
- JavaScript (ES6)
- Node.js
- npm
