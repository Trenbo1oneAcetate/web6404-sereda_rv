# Лабораторная работа №5: BookVerse

## О проекте
Интернет-магазин книг "BookVerse" с современным интерфейсом, реализованный в рамках лабораторной работы по web-программированию.

## Функциональность

### 1. Динамическая валидация формы
- Валидация полей в реальном времени
- Визуальные подсказки и сообщения об ошибках
- Проверка сложности пароля с индикатором
- Маска для номера телефона

### 2. Асинхронная отправка данных
- Отправка POST-запросов на mock-сервер
- Обработка успешной отправки и ошибок
- Модальное окно с подтверждением

### 3. Периодическое обновление данных
- Автоматическая загрузка книг каждые 5 минут
- Таймер обратного отсчёта
- Ручное обновление по требованию

### 4. Современный UI/UX
- Адаптивный дизайн
- Темная/светлая темы
- Плавные анимации
- Минималистичный стиль
