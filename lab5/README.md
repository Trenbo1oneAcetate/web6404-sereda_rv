# Лабораторная работа №5

Цель работы: изучение событийной модели JavaScript и асинхронного взаимодействия.

В работе реализована:
- динамическая валидация формы;
- отправка данных формы через POST-запрос;
- асинхронное получение данных с сервера;
- обработка ошибок;
- периодическое обновление данных.

Для имитации серверной части используется json-server.
